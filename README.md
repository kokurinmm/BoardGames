# BoardGames

Windows-приложение на C# для игры в шашки и реверси с искусственным интеллектом

## Возможности

- выбор между шашками и реверси
- алгоритм альфа-бета отсечения с настраиваемой глубиной
- алгоритм монте-карло с настраиваемым количеством симуляций (простейший вариант)
- единый интерфейс для обеих игр
