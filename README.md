# BoardGames

Кроссплатформенное приложение на C# для игры в настольные логические игры с искусственным интеллектом

## Возможности

- три игры с одинаковым интерфейсом: шашки, реверси и уголки
- алгоритм альфа-бета отсечения с настраиваемой глубиной
- алгоритм Monte Carlo Tree Search с настраиваемым временем на размышление
- режим игры без ИИ
